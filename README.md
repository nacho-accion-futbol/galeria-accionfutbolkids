# Galería Acción Fútbol

Página web profesional que muestra automáticamente galerías de fotos por fecha desde Google Drive.

## Características

✅ Lectura automática de datos desde Google Sheets  
✅ Acceso automático a carpetas de Google Drive  
✅ Galería responsiva y móvil-friendly  
✅ Actualización automática cuando se suben fotos  
✅ Diseño profesional y moderno  
✅ Ligero y rápido  

## Requisitos

- Node.js 16+
- Cuenta de GitHub (para deployar)
- Vercel (para hosting)

## Instalación local

```bash
npm install
npm run dev
```

La app estará en: http://localhost:3000

## Deploy en Vercel

1. Conecta tu repositorio GitHub a Vercel
2. Vercel automáticamente detectará Next.js
3. Agrega las variables de entorno en Vercel Settings
4. Deploy automático en cada push a main
